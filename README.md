## Features

### Supported values
- `int` (e.g. `123`)
- `bool` (e.g. `True`, `False`)
- `string` (double quotes, e.g. `"hello"`)
- `list` (non-nested, e.g. `[1, "a", 3]`)

### Supported operations
- Variable assignment: `x = 5`, `x = [1,2]`
- Variable lookup: `x`
- Built-ins:
  - `type(x)`
  - `len(x)` (only for sequence types)
  - `del x` (delete variable)

### Errors (Python-like)
- `SyntaxError`
- `IndentationError`
- `NameError`
- `TypeError`
